  • Authoring formats

    As of release 3.0, DITA-OT supports several alternative input formats in addition to standard DITA XML, including Markdown and the proposed MDITA and HDITA authoring formats currently in development for Lightweight DITA.

    • Markdown content

      Markdown is a lightweight markup language that allows you to write using an easy-to-read plain text format and convert to structurally valid markup as necessary.

    • Lightweight DITA

      DITA-OT provides preview support for the MDITA and HDITA authoring formats proposed for Lightweight DITA, or “LwDITA”. These proposed formats are alternative representations of DITA content in Markdown or HTML.

    • Markdown DITA syntax

      Markdown DITA uses CommonMark as the underlying markup language.
